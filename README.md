# CI/CD Pipeline with GitHub Actions and Kubernetes

## 🚀 Overview
This project demonstrates how to automate application deployment using GitHub Actions, Docker, and Kubernetes.

## 🧩 Client Scenario
A startup needs to reduce manual deployment errors and implement a reliable CI/CD pipeline.

## ⚙️ Tech Stack
- Python (Flask)
- Docker
- GitHub Actions
- Kubernetes

## 🔄 Pipeline Flow
1. Code push triggers pipeline
2. Docker image is built
3. Image is pushed to Docker Hub
4. (Next phase) Deploy to Kubernetes
